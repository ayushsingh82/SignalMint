# Synthesis 2026 — SignalMint

**Our project:** [**SignalMint**](md/PROTOCOL_LABS_AND_SUPERRARE_IDEAS.md) — an **autonomous market-driven art agent**. Art shaped by market signals: the agent treats auction activity (bids, bidders, speed, whale bids) as artistic input and changes the artwork in response. The market becomes the co-creator. Full autonomy loop (discover → plan → execute → verify → submit) on **Rare Protocol** with **ERC-8004** identity. Targets **Protocol Labs** (Let the Agent Cook) + **SuperRare** (Rare Protocol) bounties.

**App:** Next.js + TypeScript in `src/` — run `npm run dev` from this folder.
